- Nắm vững nền tảng AWS Cloud
- Thực hành hands-on với các dịch vụ AWS quan trọng
- Xây dựng mindset triển khai hệ thống thực tế (Production-ready)
- Hoàn thành Capstone Project cuối kỳ

## 📂 Folder Structure
Mỗi tuần gồm:
- `worklog.md`: nhật ký theo ngày
- `tasks.md`: checklist công việc + mục tiêu
- `evidence.md`: link screenshot, diagram, demo, tài liệu

## 🧾 Format Worklog
Mỗi ngày gồm:
- Nội dung học

- Output (ảnh, link, diagram)
